Application example based on vispubdata.org (accessed 9 Feb 2017, BibTeX conversion)
PubViz is a generic visualization tool for autobiographical presentation of publication lists based on a BibTeX file – this example has no customizations for vispubdata.org.
Try different prefiltered subsets: InfoVis track, InfoVis & VAST track, all tracks 2006–2015, all tracks/all years, co-authored by M.E. Gröller, co-authored by D.A. Keim
